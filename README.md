# Sorting Visualizer in JavaFX

A dynamic sorting algorithm visualizer built with JavaFX. This project is designed to help students and developers understand how different sorting algorithms work by displaying step-by-step animations of the sorting process.

## 🚀 Features

- Real-time visualization of sorting algorithms
- Adjustable animation speed
- Multiple sorting algorithms (Bubble Sort, Quick Sort, Merge Sort, etc.)
- Clean and modern UI with JavaFX

## 📚 Algorithms Planned

- Bubble Sort ✅
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort (optional)

## 🛠️ Technologies

- Java 17+
- JavaFX 21+
- Maven (optional)

## 🎯 Project Goals

- Demonstrate algorithm behavior visually
- Use clean, modular Java code
- Practice professional Git/GitHub workflows with milestones and issues

## ▶️ How to Run

Make sure you have:

- Java 17+ installed and configured as `JAVA_HOME`
- Maven installed (v3.6 or higher)

To run the application:

```bash
mvn clean javafx:run
```

*This will compile the project and launch the JavaFX visualizer.*
